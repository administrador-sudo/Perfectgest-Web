Papéis no tratamento (LGPD)
• Controlador dos dados clínicos dos pacientes: a clínica ou o profissional de saúde que utiliza o software (define finalidades e meios do tratamento no contexto da assistência).
• Operador do software: Perfect Gest Dev (PERFECT GEST DESENVOLVIMENTO DE SOFTWARE LTDA) — desenvolve e mantém a ferramenta; não decide sobre o conteúdo clínico inserido pela clínica.
• Direitos do titular (dados de saúde): dirigir-se primeiro à clínica (controlador). Suporte técnico do produto: suporte@perfectgestdev.com.

1. Natureza dos dados

O PerfectGest-Clinica III pode tratar dados pessoais sensíveis (dados de saúde) inseridos pela clínica ou profissional que utiliza o software (ex.: prontuário e histórico de consultas).

2. Papéis LGPD

• Controlador: clínica ou profissional de saúde que utiliza o software (define finalidades e meios do tratamento).
• Operador do software: Perfect Gest Dev (PERFECT GEST DESENVOLVIMENTO DE SOFTWARE LTDA).
• Suboperadores (quando aplicável):
— Microsoft (pagamentos de subscrição via Microsoft Store)
— Meta/WhatsApp (mensagens, se configurado pela clínica)
— Firebase/Google (componentes técnicos, se activados)

3. Minimização e armazenamento

• Dados clínicos permanecem no dispositivo (SQLite local).
• Logs de auditoria guardam metadados (quem / quando / o quê), sem conteúdo clínico.

4. Direitos do titular (Art. 18 LGPD)

O app oferece suporte operacional para:
• acesso e correção via ficha do paciente;
• exportação de relatório LGPD (ADMIN) em PDF/CSV (quando disponível);
• portabilidade via backup encriptado.

Para exercer direitos sobre dados de saúde, contactar primeiro a clínica (controlador).

5. Segurança recomendada (boas práticas)

• Activar BitLocker no Windows (recomendado);
• Usar senhas fortes e perfis adequados (ADMIN / DOUTOR / OPERADOR);
• Activar encriptação da base de dados no app (opcional);
• Usar backup ZIP encriptado e guardar a senha em local seguro.

6. Contato

Dúvidas técnicas sobre o produto: suporte@perfectgestdev.com
Solicitações LGPD sobre dados clínicos: contactar a clínica (controlador).
